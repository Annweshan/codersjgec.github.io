The Way We Roll Around Here

Web projects

We got plenty of web projects to work on. Jolites, LiterarySociety, Coders are to name a few.

Apps

We are just getting started now. But there are many opportunities to work on. Android apps, Firefox Os apps, etc are under development.

Localizations

What good is a software to you if it's written in some foreign language you can't understand. Localisation is as important a task as any software or Web development. Join Us in translating the web. Hindi, Bengali localizers in need.

Other Involvement
